# Zerobot

## Setup Procedure

Assuming a fresh image of Raspberry Pi 12 (Bookworm), 32-bit or 64-bit:

1. Update the system and reboot:

    ```sh
    $ sudo apt update
    $ sudo apt upgrade
    $ sudo reboot
    ```

2. Git clone this repository
3. Run install.py as root:

    ```sh
    $ sudo python3 install.py
    ```
